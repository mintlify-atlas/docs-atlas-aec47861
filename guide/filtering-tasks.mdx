## Overview

Taskflow provides a sidebar with filter options to help you focus on specific priority levels. This makes it easy to see only the tasks that matter most at any given moment.

## Using the Sidebar Filters

The sidebar on the left side of the screen contains a "Filtros" (Filters) section with four filter options:

<Steps>
  <Step title="Access the Filters">
    Look for the sidebar on the left side of the screen with the "FILTROS" heading. You'll see four filter links:
    
    - **Todas** (All)
    - **🔴 Prioridad Alta** (High Priority)
    - **🟠 Prioridad Media** (Medium Priority)
    - **🟢 Prioridad Baja** (Low Priority)
  </Step>

  <Step title="Click a Filter">
    Click on any filter link to view only tasks matching that priority level. The active filter will be highlighted to show which view you're currently in.
  </Step>

  <Step title="View Filtered Results">
    The main area will update instantly to show only the tasks matching your selected filter. Tasks that don't match will be hidden from view.
  </Step>
</Steps>

## Understanding Each Filter

### Todas (All Tasks)

<Info>This is the default view when you first open Taskflow. It shows all your tasks across all priority levels, organized into their respective sections.</Info>

Use this view when you want to:
- See your complete task list
- Get an overview of everything you need to do
- See how tasks are distributed across priority levels

### 🔴 Prioridad Alta (High Priority)

<Info>Shows only tasks marked with "Alta" (High) priority. These are your most urgent or important tasks.</Info>

Use this filter to:
- Focus on critical tasks that need immediate attention
- Plan your day by tackling high-priority items first
- Avoid getting distracted by less important tasks

### 🟠 Prioridad Media (Medium Priority)

<Info>Displays only tasks marked with "Media" (Medium) priority. These tasks are important but not urgent.</Info>

Use this filter to:
- Work on important tasks after completing high-priority items
- Plan ahead for tasks that need attention soon
- Balance your workload effectively

### 🟢 Prioridad Baja (Low Priority)

<Info>Shows only tasks marked with "Baja" (Low) priority. These are tasks that can wait or are less critical.</Info>

Use this filter to:
- Fill extra time with less urgent tasks
- Work on nice-to-have items when you have capacity
- Keep track of lower-priority items without cluttering your main view

## Visual Indicators

The active filter is highlighted in the sidebar, making it easy to see which view you're currently using. This helps you stay oriented as you switch between different priority levels throughout your day.

<Tip>Combine filters with the search function to quickly find specific tasks within a priority level.</Tip>

## Workflow Tips

- **Morning routine**: Start with the "Prioridad Alta" filter to plan your day
- **End of day**: Switch to "Todas" to see what you've accomplished
- **Planning sessions**: Use each priority filter individually to review and adjust task priorities
- **Focus time**: Filter by a single priority to avoid context switching
